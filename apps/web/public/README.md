# دليل استخدام منصة البرحاوي للدعاية والإعلان

تهانينا على إطلاق منصتك الجديدة! هذا دليل بسيط لمساعدتك في إدارة محتوى موقعك.

## 1. هيكل الموقع
يتكون الموقع من عدة صفحات رئيسية:
- **الرئيسية (index.html)**: عرض شامل للشركة.
- **من نحن (about.html)**: قصة الشركة وفريق العمل.
- **الخدمات (services.html)**: تفاصيل الخدمات المقدمة.
- **المعرض (portfolio.html)**: الأعمال السابقة.
- **المدونة (blog.html)**: المقالات التسويقية.
- **اتصل بنا (contact.html)**: بيانات الاتصال ونموذج المراسلة.
- **حجز موعد (booking.html)**: نظام جدولة الاجتماعات.

## 2. كيفية تحديث المحتوى (نظام إدارة المحتوى البسيط)
لقد قمنا بتطوير لوحة تحكم بسيطة تتيح لك تحديث البيانات دون الحاجة لخبرة برمجية:
1. افتح الملف `admin.html` في متصفحك.
2. قم بتعديل المعلومات المطلوبة (اسم الشركة، الخدمات، الأعمال، إلخ).
3. بعد الانتهاء، اضغط على زر **"حفظ التغييرات"**.
4. سيتم تحميل ملف جديد باسم `data.js`.
5. قم باستبدال الملف القديم `data.js` في مجلد الموقع بالملف الجديد الذي قمت بتحميله.
6. ستنعكس التغييرات فوراً على جميع صفحات الموقع.

## 3. المتطلبات التقنية
- الموقع يعمل بالكامل باستخدام HTML, CSS, و JavaScript.
- يعتمد على مكتبة Tailwind CSS للتصميم و Alpine.js للوظائف التفاعلية.
- الموقع متجاوب بالكامل مع جميع الشاشات.
